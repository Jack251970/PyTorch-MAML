# How to run Pytorch-MAML

Create conda environment with Python 3.12:
```bazaar
conda create -n Pytorch-MAML python=3.12
conda activate Pytorch-MAML
```

Install Pytorch:
```bazaar
pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu130
```

Install other dependencies:
```bazaar
pip3 install tqdm tensorboardx scipy pandas scikit-learn
```
